Tidak cukup ruang pada partisi

Coba ubah hosts file tujuan di preferensi menjadi /data/data/hosts dan terapkan AdAway kembali.

Does not work on Android 4.4+

Try changing the target hosts file in preferences from /data/data/hosts to /data/hosts or /system/etc/hosts and apply AdAway again.

iklan tidak dapat di blok pada aplikasi XYZ!

Beberapa hostnames mungkin hilanf pada hosts yang di sediakan dari Hosts Sources atau aplikasi telah memasukkan gambar untuk menampilkan iklan tanpa koneksi internet.

Kamu dapat mencatat permintaan DNS (Menu->Log DNS Requests) dari AdAway untuk mengetahui nama host tambahan yang harus diblokir.

Add the suspicious hostnames to your own Blacklist by long pressing the entries in the Log and report these hostnames when you have verified them in the Forum Hosts Inbox of hosts-file.net.

Iklan tembolok

Kadangkala aplikasimelakukan cache iklan setelah diunduh. Ini menyebabkan iklan yang tersisa pada beberapa aplikasi. Kamu bisa mencoba menghapus cache untuk aplikasi ini dalam daftar Aplikasi Android untuk menghindari masalah ini.

Aplikasi XYZ tidak lagi berfungsi!

Beberapa aplikasi membutuhkan komunikasi dengan hostname yang diblok oleh AdAway atau menolak untuk berfungsi ketika hostname yang seharusnya menyajikan iklan tidak dapat dijangkau. Lihat https://github.com/dschuermann/ad-away/wiki/ProblematicApps untuk mendapatkan daftar aplikasi yang diketahui memiliki masalah. Jika tidak, cari tahu hostname mana yang diperlukan dan tambahkan ke dalam Whitelist dibawah Your Lists dan laporkan ke bug tracker of AdAway.

Tombol kembali pada web browser tidak berfungsi

Anda dapat mengaktifkan webserver lokal AdAway di preferensi sebagai solusi.
