Perangkat HTC

AdAway tidak bekerja ketika Anda menggunakan perangkat dengan apa yang disebut S-ON. 'Fitur' ini ada pada banyak perangkat HTC dan mencegah AdAway dari menulis file host.

S-ON/S-OFF?

S-OFF berarti bahwa bagian NAND pada perangkat tidak terkunci dan dapat ditulis. Pengaturan standar untuk perangkat HTC adalah S-ON, yang berarti bahwa tidak dapat mengakses area tertentu dari sistem juga tidak bisa menjamin root permanen. Selanjutnya, memeriksa tanda tangan untuk gambar firmware juga dijamin oleh S-ON flag.

Apakah saya harus S-ON atau S-OFF?

Boot into the Boot Menu pada perangkat anda dengan menekan tombol volume down bersamaan dengan tombol power dan text diatas akan muncul sebagai status S-OFF or S-ON. Full root umumnya berati S-OFF.‚èé

Solusi

Prasyarat: Anda harus menginstal SDK Android dengan ADB shell pada PC Anda.

  1. Boot into the Boot Menu pada perangkat anda dengan menekan tombol volume down bersamaan dengan tombol power.
  2. Gunakan tombol volume down untuk memilih recovery.
  3. Pada volume pemulihan jarum jam ke "partisi menu" dan tekan trackpad untuk memilih
  4. Pilih mount /system, mount /sdcard and mount /data.
  5. Tancapkan usb anda dan buka command line pada pc anda.
  6. Masuk adb shell dan tulis
    ln -s /data/data/hosts /system/etc/hosts
    (Ini membuat sebuah simbol link, yang memungkinkan AdAway untuk mengedit host file yang disimpan dalam /data sementara mengizinkan OS untuk menggunakan file seolah-olah itu disimpan dalam /system.)
  7. Reboot perangkat Anda dan set Sasaran file hosts untuk /data/data/hosts dalam preferensi AdAway.
  8. AdAway seharusnya bekerja sekarang.
