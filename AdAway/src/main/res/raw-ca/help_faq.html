Ingressos per publicitat

Sovint es diu que, si un usuari no clica els anuncis, els desenvolupadors no reben ingressos. Això és fals, ja que també es reben ingressos mostrant els anuncis. No obstant això, mostrar o amagar els anuncis es decisió vostra.

Com funciona AdAway?

AdAway utilitza el fitxer hosts per bloquejar noms de màquina que serveixen publicitat. Aquest fitxer es pot trobar a /system/etc/hosts i és un mètode tradicional per associar noms de màquina a adreçes IP sense dependre dels servidors DNS. Tots els noms de màquina que es volen bloquejar es redireccionen a 127.0.0.1, que significa que es redireccionen al vostre propi dispositiu.

Perquè està desactivat l'ajustament de trobar actualitzacions diàriament?

Només es poden trobar actualitzacions quan AdAway està instal·lat a la memòria interna del dispositiu. Això és una restricció de la plataforma Android.

Per què haig de reiniciar Android perquè els canvis facin efecte?

El Java d'Android manté la seva llista d'entrades DNS internament. El sistema utilitzarà el nou arxiu hosts immediatament (ho podeu comprovar amb "ping") però, no obstant, haureu de reiniciar Android per reinicialitzar la llista d'entrades de DNS internes.

Com s'utilitza el servidor web d'AdAway?

AdAway també funciona sense el servidor web!⏎

Podeu activar el servidor web local als ajustaments d'AdAway per respondre peticions a la vostra adreça IP local 127.0.0.1. Això significa que les peticions per anuncis que són redirigides a 127.0.0.1 són respostes pel servidor web d'AdAway.⏎
Algunes aplicacions no funcionen si no poden contactar els servidors d'anuncis i, amb aquest mètode, mostraran contingut en blanc.

Com puc bloquejar noms de màquina específics?

Add the hostnames you want to block to the Blacklist (=Black) under Your Lists. Additionally, hostnames you want to exclude from blocking can be added to the Whitelist (=White) and hostnames you want to redirect to a specific IP address belong to the Redirection List (=Redirection).
