Aparelhos HTC

O AdAway não funciona quando utiliza um aparelho com S-ON. Esta 'característica' existe em muitos aparelhos HTC e impede a escrita no ficheiro de hosts.

S-ON/S-OFF?

S-OFF significa que a partição NAND do aparelho está desbloqueada e pode ser escrita. A definição por defeito nos aparelhos HTC é S-ON, o que significa que nem pode aceder a algumas áreas do sistema nem pode garantir um root permanente. Além disso, a verificação da assinatura nas imagens de firmware é também verificada pelo S-ON.

Tenho S-ON ou S-OFF?

Entre no Boot Menu no seu aparelho pressionado volume para baixo enquanto pressiona o botão de ligar e no topo irá aparecer uma mensagem que informa se se encontra com S-OFF ou S-ON. Um root completo normalmente significa S-OFF.

Mais informação em www.addictivetips.com.

Alternativa

Pré-requisito: Terá que instalar um Android SDK com ADB shell no seu PC.

  1. Entre no Boot Menu no seu aparelho pressionado volume para baixo enquanto pressiona o botão de ligar.
  2. Utilize o botão volume para baixo para escolher recovery.
  3. Na Clockwork Recovery, escolha "partitions menu" e clique para seleccionar
  4. Seleccione mount /system, mount /sdcard e mount /data.
  5. Ligue o cabo USB e abra uma linha de comandos no PC.
  6. Entre na adb shell e escreva ln -s /data/hosts /system/etc/hosts (Isto irá criar uma ligação simbólica que permitirá o AdAway editar o ficheiro de hosts armazenado em /data permitindo simultaneamente que o SO utilize o ficheiro como se estivesse armazenado em /system.)
  7. Reinicie o aparelho e defina o Ficheiro de hosts de destino para /data/hosts nas preferências do AdAway.
  8. Agora o AdAway deverá funcionar.
