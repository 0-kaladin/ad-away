Rendimentos pelos anúncios

Existe uma falsa ideia de que se um utilizador nunca clicar nos anúncios, bloqueá-los não afectará um site ou aplicação financeiramente. É uma ideia errada - os programadores também ganham dinheiro apenas por mostrar os anúncios. Bloquear ou não os anúncios é consigo. Pessoalmente não uso aplicações com anúncios porque são muito irritantes. Desta forma, sem um bloqueador de anúncios, não instalarei estas aplicações.

Como funciona o AdAway?

O AdAway utiliza o ficheiro de hosts para bloquear hostnames que mostrem anúncios. Este ficheiro poderá ser encontrado em /system/etc/hosts que mapeia hostnames para endereços de IP. É uma forma tradicional de traduzir hostnames em IPs sem utilizar os Sistemas de Nomes de Domínio (DNS). Todos os hostnames indesejados serão redireccionados para 127.0.0.1 o que significa que apontarão para o seu próprio aparelho.

Qual o porquê de verificação diária de actualizações estar desactivada?

A verificação automática de actualizações apenas está disponível quando o AdAway se encontra instalado na memória interna. Isto deve-se a restrições da própria plataforma Android.

Porque tenho de reiniciar o Android para que as alterações surtam efeito?

O Java no Android possui a sua própria cache de DNS. Este sistema operativo começará a ler automaticamente o novo ficheiro de hosts (verifique isto com um ping na linha de comandos), mas será necessário reiniciar o aparelho para que a cache DNS do Java seja reconstruída.

Como utilizar o webserver no AdAway?

O AdAway também funcionará sem utilizar o webserver!/n

Poderá activar um webserver local nas preferências do AdAway para responder a pedidos no IP local 127.0.0.1. Isto significa que pedidos de ligação a servidores de anúncios por parte das aplicações, que são redireccionados para 127.0.0.1 serão respondidos pelo webserver do AdAway./n
Algumas aplicações recusar-se-ão a funcionar quando não conseguirem contactar os servidores de anúncios. Com este método ficarão novamente "contactáveis", respondendo com uma página em branco sem imagens.

Como posso bloquear/desbloquear alguns hostnames em particular?

Adicione os hostnames que pretende bloquear à Blacklist n'As suas listas. Adicionalmente, os hostnames que pretenda parar de bloquear poderão ser adicionados à Whitelist bem como poderá redireccionar para um IP específico os hostnames que pretenda, acedendo para isso à Lista de Redireccionamentos.
