Inte tillräckligt med utrymme på partitionen

Try changing the target hosts file in preferences to /data/data/hosts (or /data/hosts) and apply AdAway again.

Does not work on Android 4.4+

Try changing the target hosts file in preferences from /data/data/hosts to /data/hosts or /system/etc/hosts and apply AdAway again.

Den blockerar inte annonser i applikationen XYZ!

Vissa värdnamn kan saknas i de givna hostfilerna från Värdkällor eller att applikationen har annonsbilderna inbyggt i sig själva för att kunna visa reklam utan tillgång till internet.

Du kan logga DNS förfrågningar (Meny->Logga DNS Förfrågningar) från AdAway för att hitta ytterligare värdnamn att blockera.

Lägg till suspekta värdnamn till din egna Svarta Listan genom att långtrycka på inlägg i loggen och rapportera dessa värdnamn när du har verifierat dem i Forumet Hosts Inbox av hosts-file.net

Cachad Reklam

Ibland så cachar applikationer sin reklam efter att ha hämtats hem. Detta leder till att reklamen finns kvar i vissa applikationer. Du kan testa att ta bort cachen från applikationen i Androids Applista för att kringgå detta problem.

Applikation XYZ fungerar inte längre!

Vissa applikationer behöver kommunicera med ett värdnamn som är blockerat av AdAway eller vägrar att fungera när värdnamnet som skulle ge reklam inte kan nås. Se https://github.com/dschuermann/ad-away/wiki/ProblematicApps för att få en lista av kända applikationer som har problem. Annars, ta reda på vilka värdnamn som behövs och lägg till dem till din Vitlista under Dina Listor och rapportera dem till bug tracker för AdAway.

Bakåtknappen i webbläsaren fungerar inte

Du kan slå på AdAways lokala webbserver i inställningarna som en tillfällig lösning.
