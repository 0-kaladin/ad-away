Einkommen durch Werbung

Es besteht eine verbreitete ,falsche, Auffassung, das wenn ein Nutzer nie auf Werbung klickt, dies einer Webseite oder Anwendung finanziell nicht schaden könnte. Falsch ist dies insofern deswegen weil Entwickler alleine durchs Anzeigen von Werbung Geld verdienen. Es ist deine Entscheidung ob du Werbung blockierst oder nicht. Ich persönlich würde keine Anwendungen verwenden die Werbung anzeigen, weil mich diese äußerst stören. Ohne einem Werbungblockierer würd ich diese Anwendungen nicht installieren.

Wie funktioniert AdAway?

AdAway benutzt die Hosts Datei um Hostnamen zu blockieren, die Werbung zustellen. Die Hosts Datei, ist eine Datei, die sich in /system/etc/hosts befindet und Hostnamen in IP Adressen übersetzt. Dies ist die ursprüngliche Art Hostnamen zu IP Adressen Paare zu definieren, ohne vom Domain Name System (DNS) gebrauch zu machen. Alle unerwünschten Hostnamen werden zu 127.0.0.1 umgeleitet. Diese führt zum eigenen Gerät.

Warum muss ich Android neu starten, um die Änderungen wirksam werden zu lassen?

Java auf Android hält seine eigenen, internen DNS Puffer. Das Betriebssystem übernimmt die neue Hosts Datei sofort (dies kann man mit einem Ping über die Kommandozeile überprüfen). Jedoch ist es nötig Android neuzustarten um den DNS Puffer zu erneuern.

Wie verwendet man den Webserver in AdAway?

AdAway funktioniert auch ohne einen Webserver aufzusetzen !

Sie können einen lokalen Webserver in den Einstellungen von AdAway aktivieren, um Requests an die lokale IP Adresse 127.0.0.1 zu beantworten. Das bedeutet das Requests von Anwendungen zu Ad Servern, welche zu 127.0.0.1 umgeleitet werden von AdAways Webserver beantwortet werden.
Manche Anwendungen funktionieren nicht wenn ad Server nicht erreichbar sind. Mit dieser Methode werden sie wieder erreichbar, antworten mit einer leeren Seite und ohne Werbung.

Die Werbung in App XYZ wird nicht blockiert!

Einige Hostnamen mögen in der mitgelieferten Hostdatei Host Quellen fehlen oder die Anwendung bringt Bilder mit um Werbung ohne Internetzugriff darstellen zu können.

Sie können DNS Anfragen (Menü->Log DNS Requests) loggen um zu sehen welche zusätzlichen Hostnamen sie noch blocken wollen.

Fügen sie die verdächtigen Hostnamen zu ihrer eigenen Blacklist hinzu. Dies geschieht indem sie längere Zeit auf die Einträge in der log Datei drücken und melden sie diese Hostdateien, nachdem sie sich von ihrer Richtigkeit im Forum überzeugt haben Hosts Inbox of hosts-file.net.

Applikation XYZ funktioniert nicht mehr richtig!

Manche Anwendungen müssen mit einem Hostnamen kommunizieren, der von AdAway blockiert wird oder nicht funktioniert wenn Hostnamen die Werbung liefern nicht erreichbar sind. Schauen sie auf http://code.google.com/p/ad-away/wiki/ProblemAppsum eine Liste von Anwendungen zu erhalten, die bereits gemeldet sind in diesem Fall Probleme zu haben. Anderseits folgen sie den bereits gegebenen Anweisungen und fügen sie diese zu ihrer Whitelist unter Ihre Listen hinzu und melden sie diese dem Bug Tracker von AdAway .

Wie kann ich bestimmte Hostname sperren/entsperren?

Sie können ihren eigenen Server aufsetzen, um eine Hostsdatei, mit Hostnamen die sie blockieren wollen, zur Verfügung zu stellen. Fügen sie diese zu Hosts Sources hinzu oder fügen sie die Hostnamen, die sie blockieren wollen, zur Blacklist (=Schwarz) unter Your Lists hinzu. Zusätzlich können Hostnamen die sie nicht blockieren wollen zur Whitelist (=White) hinzugefügt werden und Hostnamen die sie zu einer spezifischen IP Adresse umleiten wollen gehören zur Redirection List (=Umleitung).
