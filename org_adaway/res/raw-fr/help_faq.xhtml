Revenu par la publicité

Il y a une idée fausse souvent déclarée que si un utilisateur n'a jamais cliquer sur les annonces, puis les bloque ne sera pas mal à un site ou une application financière. Ceci est faux, les développeurs gagnent de l'argent simplement en affichant des annonces. C'est à vous de décider si vous bloquez des annonces ou non. Personnellement, je ne voudrais pas utiliser des applications qui affichent des annonces, car je les trouve très ennuyeux, sans un bloqueur de publicité, je n'aurai pas installé ces applications.

Comment AdAway fonctionne ?

AdAway utilise le fichier hosts pour bloquer des noms d'hôtes, qui donnent les publicités. Le fichier hosts est un fichier qui se trouve dans /system/etc/hosts qui associe les noms d'hôtes à des IP. C'est une façon traditionnelle de définir le nom d'hôte à des paires d'adresses IP sans passer par le Domain Name System (DNS). Tous les noms d'hôtes indésirables sont redirigés vers 127.0.0.1 qui signifie qu'ils pointent vers votre propre appareil.

Pourquoi dois-je redémarrer Android pour que les changements prennent effet ?

Java sur Android maintient son propre cache DNS interne. Le système d'exploitation reflète le nouveau fichier hosts immédiatement mais vous aurez besoin de redémarrer Android pour reconstruire le cache DNS de Java.

Comment utiliser le serveur web dans AdAway?

AdAway fonctionnera également sans utiliser le serveur web !

Vous pouvez activer un serveur web local dans les préférences de AdAway afin de répondre aux demandes d'adresses IP locales comme 127.0.0.1. Cela signifie que les demandes des applications de serveurs de publicité sont redirigés vers 127.0.0.1 ou d'autres adresses IP locales traitées par le serveur de AdAway.
Certaines applications refusent de fonctionner, lorsque les serveurs de pubs ne sont pas accessibles. Avec cette méthode, ils seront accessibles à nouveau, affichant une page blanche et aucune image publicitaire.

Les pubs dans AngryBirds sont encore affichées !

Supprimer le répertoire burstlyImageCache sur votre carte SD. Il est utilisé comme un cache annonce pour Angry Birds.

Cela ne bloque pas les publicités dans l'application XYZ !

Certains noms d'hôtes peuvent manqués dans les fichiers hosts fournis par les sources Hosts ou la demande a fourni les images pour fournir des annonces, sans accès à l'internet. Il y a deux méthodes pour trouver des noms d'hôtes qui doivent être bloqués :

• Ouvrir /data/data/org.example/databases/webviewCache.db, où org.example est le nom du package de l'application qui affiche des pubs, et chercher des noms d'hôtes qui semblent être utilisés pour diffuser des annonces et ne sont pas utilisés autrement dans l'application.

• Installer Wireshark sur votre ordinateur et acheminer tout le trafic d'Android via votre ordinateur, qui est utilisé comme un proxy, et sniffer les demandes que cette application fait. Comme alternative sur Android, vous pouvez utiliser 'Shark pour Root' avec 'Shark Reader'

Ajouter les noms d'hôtes à votre propre liste noire comme détaillé ci-dessous et rapporter ces noms d'hôtes quand vous les avez vérifié dans le Forum Hosts Inbox des hosts-file.net.

L'application XYZ ne fonctionne plus !

Certaines applications ont besoin de communiquer avec un hôte qui est bloqué par AdAway ou refuse de fonctionner quand les noms d'hôtes qui devrait afficher les pubs ne sont pas accessibles. Voir http://code.google.com/p/ad-away/wiki/ProblemApps pour obtenir une liste des applications connues qui ont des problèmes. Sinon faites comme décrit précédemment pour savoir quels noms de machines sont nécessaires et les ajouter à votre liste blanche sous vos listes et de les signaler aubug tracker d'AdAway.

Comment puis-je bloquer/débloquer des noms d'hôtes spécifiques ?

Vous pouvez configurer votre propre serveur fournissant un fichier hosts avec des noms d'hôtes que vous souhaitez bloquer et l'ajouter aux sources d'hôtes ou d'ajouter les noms d'hôtes que vous souhaitez bloquer dans la liste noire (= noir) sous vos listes. En outre, les noms d'hôtes que vous voulez exclure du blocage peuvent être ajouté à la liste blanche (= blanc) et des noms d'hôtes que vous souhaitez rediriger vers une adresse IP spécifique appartiendront à la Liste de redirection (=Redirection).
